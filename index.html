Your mandate

Two of the nine criteria — Competence Fit and Portfolio Fit — are judged only against what you enter here. Leave a field blank and that dimension is marked Not assessable rather than guessed. Nothing here is pre-filled with invented data.

Benchmark to beat — S&P 500 total return

The trailing 10-year figure is the live hurdle and refreshes about monthly when you open the app (an artifact can't run while closed — a true unattended monthly fetch belongs on a backend). The long-run figure is a stable anchor. Set a custom hurdle below only if you want to override the S&P 500.

Weights

The composite is a weighted blend of the nine criteria. Tune to taste; the total need not equal 100 — weights are normalised automatically, and re-normalised across only the criteria that could actually be assessed.

Scored to date

History is stored in this browser on this device — it does not sync between your phone and laptop, and clearing site data erases it. To move or back up evaluations: Export JSON on one device, then Import JSON on the other. Importing merges by entry, so re-importing the same file adds nothing and never overwrites what's already here.

How the read is built

Every opportunity is scored 0–10 on nine criteria a savvy allocator would insist on. The model scores only what your input actually evidences; the maths, the verdict, and the confidence are computed here in the open — not by the model — so nothing is a black box.

The two dials

Conviction is the weighted composite. Evidence coverage is how much of the framework your input let it assess. A high conviction on thin coverage is drawn deliberately hollow — a savvy investor distrusts a confident score built on unknowns. Below 35% coverage, the verdict is withheld as Insufficient data no matter how the numbers land.

Zero fabrication. Missing figures are returned as Not stated and lower coverage — never filled with an industry-average guess. Competence and portfolio fit come only from your Playbook.
Connection

Sharkfolio calls the Anthropic API directly from your browser with your own key. The key is stored only in this browser, on this device — it is never sent anywhere except Anthropic, and it is not baked into the app, so anyone opening the site without their own key simply can't run a score.

Create one at console.anthropic.com → API Keys. Usage is billed to that account.

Use any model string valid for your account (e.g. claude-sonnet-5, claude-opus-4-8). Confirm the exact name in your Anthropic console. Web-search benchmark refresh needs a model and key with web search enabled.
